glossary

Fork betyder:

En Fork är en kopia av ett repository. Att Forka ett repository menas att kopierar ett repository för att jobba fritt med koden som finns utan att influera den primära koden. Denna nya repositoryn man har forkat tillhör inte längre den som har skapad den, den tillhör den som har forkat det. Fork används för att ändra ett projekt som inte tillhör dig eller för att ha någon annans projekt som startpunkt för att lägga till sina egna ideer. Man kan också dela ett repository för att skapa en kopia för att experimentera utan att förstöra den primära repositoryn.s


Commit betyder:

Commit är när man spara en en individuell ändring i din repository. Dessa ändringarna lagras så att andra deltagare kan se vad som har ändrats och när. När man gör en kommit måste man lämna ett meddelande för att förklar vad som har ändrats för att göra det lättare för dem andra deltagarna.
